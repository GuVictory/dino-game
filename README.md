# dino-game



## Что это такое?

Это репозиторий, в котором живет игра с бегущим динозвариком

## Как запустить?

Для запуска проекта нужно:

1. Склонировать репозиторий
2. `npm install`
3. `npm run dev`


## Как написать такое с нуля?
